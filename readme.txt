CountDownDays - Plugin for counting down the event, simple configuration and widgetable (drag and place).

== Description ==

CountDownDays - Plugin for counting down the event base the day. CountDownDays plugin is widgetable at your sidebar. Simple configuration, simple view, and friendly to every user. 

If any problems or feedback please contact me [www.nulis.web.id](http://www.nulis.web.id "Contact me") . The readme in the Indonesian language can be read in http://arisnb.nulis.web.id/count-down-days.html

Thanks to Purwedi Kurniawan.



== Installation ==

1. Download file 
2. Upload and extract in plugin folder, or upload by add new plugin at your WordPress Dashboard
3. Activate the plugin through the 'Plugins' menu in WordPress
4. Configure/Setting through your WordPress Dashboard : Settings >> Count Down Days
5. Add to your widgets : Appearance >> Widgets. (Drag and place at your widgets)
6. Enjoy it.
